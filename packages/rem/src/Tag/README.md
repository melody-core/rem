# Button

## How to use

```jsx
import { Tag } from 'my-awesome-library';
<Tag onClick={handlerOnClick}>Hello Button</Tag>;
```

## Props

- **className** - class applied to root `Tag` html element.

| propName  | propType | defaultValue | isRequired |
| --------- | -------- | ------------ | ---------- |
| className | string   |              | no         |
